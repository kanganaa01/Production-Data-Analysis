# Production-Data-Analysis-Dashboard



## Problem Statement

This dashboard helps organizations monitor and optimize their production performance by analyzing demand, availability, supply shortages, profit, and losses. It enables decision-makers to identify gaps between demand and supply, understand operational inefficiencies, and improve business planning.

By tracking daily production metrics and financial impact, the dashboard supports data-driven decisions to reduce shortages, minimize losses, and improve profitability.

---

## Steps followed

* Step 1 : Production dataset was collected and structured into tables containing demand, availability, supply, profit, and loss information.

* Step 2 : Data was imported into Power BI Desktop.

* Step 3 : Power Query Editor was used to check column distribution, quality, and profile to ensure accurate data formatting.

* Step 4 : Date columns were formatted properly to enable time-based analysis.

* Step 5 : Measures were created to calculate:

  • Average Demand per Day
  • Average Availability per Day
  • Total Supply Shortage
  • Total Profit
  • Total Loss
  • Average Daily Loss

* Step 6 : KPI card visuals were added to display key performance indicators clearly.

* Step 7 : Filters were added for Order Date and Product Name to allow dynamic analysis.

* Step 8 : Dashboard theme, background visuals, and layout were customized for better readability and presentation.

* Step 9 : The report was published to Power BI Service for sharing and monitoring.

# Insights

A multi-section report was created on Power BI Desktop and published to Power BI Service.

Following inferences can be drawn from the dashboard;

### [1] Demand vs Availability

• Average demand per day shows market requirement trends.
• Average availability per day highlights production capacity.
• Supply shortage indicates the gap between production and demand.

Thus, monitoring shortages helps improve planning and inventory management.



### [2] Financial Performance

• Total profit represents successful production efficiency.
• Total loss highlights operational inefficiencies or supply issues.
• Average daily loss helps track recurring financial impact.

Thus, reducing shortages and inefficiencies can directly improve profitability.


### [3] Operational Insights

• Date filters allow tracking performance trends over time.
• Product filters help identify underperforming products.
• KPI visuals provide quick performance monitoring for decision-makers.

---

# Conclusion

This Production Data Analysis dashboard converts operational data into actionable insights through visual KPIs and interactive filters. It helps businesses understand production efficiency, reduce losses, and optimize resource allocation.

Future enhancements may include predictive demand forecasting, inventory optimization, and supplier performance analysis.
